## Description du projet
**LinkUp** est une application **full stack** qui s’appuie sur une base de données **MongoDB ET PostgresSQL** , une **API Spring Boot (Java)** , **Next.js**  , **React** et pleins d'autres fonctionalitées afin d’offrir une expérience complète sur le web.

Depuis notre arrivée au cégep, nous avons remarqué que l’un des plus grands défis pour les étudiants est de **trouver un temps commun** afin de se rencontrer et de **communiquer efficacement**. Au cégep, contrairement au secondaire où les horaires sont souvent similaires, chaque étudiant possède un emploi du temps différent. Il faut donc s’adapter à une réalité où l’organisation devient essentielle.

Pour répondre à ce besoin, nous développerons une application qui **connecte les étudiants** (peu importe le cégep) afin de **repérer des disponibilités communes** et de planifier des moments pour **étudier**, faire des **activités**, ou tout autre projet.

Nous prévoyons également d’intégrer une section **Canvas** permettant d’interagir autrement, notamment via des **éléments visuels** (images et cartes interactives) que les utilisateurs pourront **placer** et **manipuler** dans un espace collaboratif.

## CHOIX DES TECHNOLOGIES ##
Ce projet représente un nouveau niveau pour nous. Même si moi (Saad) j’ai une légère expérience avec React et TypeScript, je vais m’adapter afin que mes coéquipiers et moi puissions utiliser les meilleures technologies et les meilleurs frameworks, entièrement open source et gratuits et tout en restant accessibles et familiers pour l’équipe.Nous voulons éviter de devoir apprendre un langage complètement nouveau (comme le C++), afin de rester efficaces, d’avancer plus rapidement, et de garder une bonne cohésion de travail. Cette décision convient également à mes coéquipiers et leur plaît.

### Liste des Technologies ###

**Spring :**  
Spring est un framework Java qui permet d’agir comme un serveur **stateless**. Ça veut dire que le programme tourne toujours et répond aux requêtes, sans garder d’état entre chaque requête (un peu comme notre serveur JavaScript de la session dernière). Le serveur reste toujours ouvert tant que l’application est en exécution.

**Spring Boot :**  
Spring Boot est un sous-framework de Spring qui ajoute plein de fonctionnalités prêtes à l’emploi (auto-config, serveur intégré, ORM, etc.).

**Technologies :**
- (Java) Spring Boot + JPA (pour la partie SQL) ORM 
- (Java) Spring Boot + MongoDB Driver / Spring Data MongoDB (pour la partie NoSQL) (API / logique)


### Front-end stack ###

- Nous allons utiliser **React** pour développer le front-end. Ce sera notre “moteur” principal.
- Comme framework de liaison/routing, nous allons utiliser **Next.js**.
- Pour le design, nous allons utiliser **shadcn/ui** et/ou **Tailwind CSS** *(à confirmer)*.
- La mise en place peut être assez complexe.
